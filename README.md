# FNF Azure Engine
simple FNF engine build on CTF2.5
cuz Clickteam is much easier :)
Current latest version:  0.1b

# Features
- Simple input system (currently ghost tapping)
- Custom keybinds
- 3 Example songs
- In-game editor (it sucks)
- Original FNF-like menus
